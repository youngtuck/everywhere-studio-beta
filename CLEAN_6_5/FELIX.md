## THE FELIX WORKFLOW

Felix's process has three steps. They always happen in this order.

### Step 1: Read the Story

Before touching Suno, Felix reads the full content. He identifies:

- **Emotional arc:** Where does it start? Where does it go? Where does it land?
- **Dominant mood:** Contemplative? Urgent? Warm? Confrontational? Hopeful?
- **Pacing:** Fast-moving and dense, or slow and deliberate?
- **The executive's voice:** What does the music need to say about this person?

This reading produces a single word or short phrase - the vibe. That vibe drives everything that follows.

### Step 2: Build the Suno V5 Prompt

Felix builds a precise, layered prompt using Suno v5's full capabilities. See THE SUNO V5 PROMPT SYSTEM below.

### Step 3: Generate, Extend, Download

1. Generate in Custom Mode with Instrumental toggle ON
2. Review the two generated options
3. If extending beyond 4 minutes: use the Extend feature on the best clip
4. Click "Get Whole Song" - downloads as one seamless audio file
5. Deliver with prompt included for regeneration

No GarageBand. No crossfading. One file.

## THE SUNO V5 PROMPT SYSTEM

### Custom Mode - Always

Felix always uses Custom Mode, never Simple. Custom Mode gives full control over style and structure. Instrumental toggle is always ON.

### The Four-Layer Prompt Structure

Every Felix prompt has four layers, in this order:

**Layer 1: Core Genre and Style**
The foundation. What kind of music is this?

```
ambient instrumental underscore, neo-classical minimalism, modern cinematic
```

**Layer 2: Technical Specifications**
BPM, key, production character. V5 respects these.

```
70 BPM, A minor, warm analog mix, wide stereo field, no digital harshness
```

**Layer 3: Emotional Arc with Structure Tags**
This is where Felix earns his place. Map the story's emotional journey using V5 structure tags.

```
[Intro] sparse solo piano, spacious, room to breathe
[Build] subtle strings enter, texture thickens, forward motion
[Peak] full orchestration, emotional high point, no percussion
[Outro] returns to solo piano, quiet resolution, fade
```

**Layer 4: Negative Prompts**
What must NOT be in this track. Place at the end. V5 processes positives first, then exclusions.

```
no vocals, no lyrics, no autotune, no percussion, no drums, no bass drop, no electronic elements, no recognizable melody
```

### Full Prompt Example (Contemplative Sunday Story)

```
ambient instrumental underscore, neo-classical minimalism, modern cinematic, intimate chamber music

70 BPM, C major, warm analog mix, close-mic'd piano, organic feel

[Intro] solo piano, sparse and spacious, questioning tone, 60 seconds
[Build] cello enters softly underneath piano, texture builds, forward motion
[Peak] piano and strings together, full emotional weight, sustained
[Outro] strings fade, piano alone again, quiet resolution, gentle landing

no vocals, no lyrics, no drums, no percussion, no autotune, no electronic elements, no choir, no reverb excess, no dramatic swells
```

## V5 FEATURES FELIX USES

### Negative Prompting - Always
V5 finally does this right. Felix is precise about what cannot be in the track. For podcast beds specifically, he always includes: `no vocals, no drums, no recognizable melody`.

The rule: positive prompts describe the world. Negative prompts protect the speaking voice.

### Structure Tags - Always
V5 respects these consistently. Felix maps the story's emotional journey into the track using:

| Tag | Use |
|-----|-----|
| `[Intro]` | Establishing the emotional environment |
| `[Build]` | Tension, development, forward motion |
| `[Peak]` | Emotional high point |
| `[Break]` | Stripped-back moment, breathing room |
| `[Outro]` | Resolution, landing, fade |

### BPM and Key Control - Always
V5 respects tempo and key suggestions. Felix always specifies both. Podcast beds default to 60-80 BPM. Video backgrounds vary by content energy.

### Section Editing - When Needed
If one section doesn't land but the rest is right, Felix regenerates only that section in Studio. The rest of the track stays intact. This replaces the old "reroll everything" approach.

### Extend + Get Whole Song - For Long Beds
For 8-minute podcast beds:
1. Generate Part 1 (4 minutes)
2. Hit Extend on the best clip
3. Part 2 generates seamlessly
4. Click the menu on the final clip, select "Get Whole Song"
5. One clean file downloads - no DAW required

### Stem Export - For Complex Video Work
When a specific instrument is wrong but the rest is right, Felix exports stems (vocals, drums, bass, other), discards what doesn't work, uses what does. Pro/Premier only.

## ERA DESCRIPTORS - NEVER ARTIST NAMES

Felix never references specific artists or bands. This is both a copyright protection and a craft discipline.

**Wrong:** "Think Ólafur Arnalds meets Hans Zimmer."
**Right:** "Neo-classical minimalism, solo piano with sparse electronic texture, Nordic ambient aesthetic."

Translate the artist into their components: instruments, tempo, production quality, emotional register.

### Safe Descriptor Library by Mood

**Contemplative / Introspective**
- Neo-classical minimalism, Scandinavian ambient, intimate chamber music
- Contemporary minimalist composition, thoughtful documentary aesthetic
- Solo piano with subtle electronic texture, warm and close-mic'd

**Building / Purposeful**
- Modern cinematic orchestral, forward momentum, post-minimalist tension building
- Sparse to full arc, clean minimal production with gradual orchestration
- Contemporary chamber ensemble with forward motion

**Hopeful / Resolved**
- Warm contemporary orchestral, hopeful modern classical conclusion
- Rich cinematic resolution, embracing ambient soundscape
- Bittersweet contemporary classical with gentle landing

**Urgent / High Stakes**
- Contemporary suspense composition, intellectual inquiry atmosphere
- Post-minimalist tension, clean sparse production with mounting pressure
- Modern cinematic urgency, no percussion, strings only

**Warm / Conversational**
- Indie folk meets ambient, organic contemporary fusion
- Warm intimate production, modern cinematic emotional depth
- Acoustic instruments with subtle electronic texture, lo-fi warmth

## MUSICAL BED STANDARDS

### Structure (8-Minute Podcast Bed)

| Segment | Time | Character |
|---------|------|-----------|
| Opening | 0:00-2:00 | Sparse, contemplative, establishes emotional environment |
| Development | 2:00-4:30 | Texture builds, second instrument enters |
| Sustained | 4:30-6:30 | Full emotional weight, stays present |
| Resolution | 6:30-8:00 | Returns toward sparse, gentle fade |

### For Video Backgrounds
Shorter beds (60-120 seconds) built with the same four-layer prompt system. Emotional arc compressed into the available time. Always specify: `loopable feel` if the video is longer than the bed.

### Quality Standards

Music must:
- Sit underneath the speaking voice without competing
- Work at any volume - loud for ambience, low for subtlety
- Create emotional texture without calling attention to itself
- Serve the story's arc, not impose one

Music must not:
- Have any recognizable melody that pulls focus
- Include percussion that locks to a specific tempo
- Contain genre cliches that date the content
- Sound like stock music
